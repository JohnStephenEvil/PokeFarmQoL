Custom Search

Here you can custom find any Pokémon you want! Hover over "Custom Search Help" for more info.

Custom Search Help Custom search by Pokémon name

Select Search on items and/or Custom Pokémon and type the name of the Pokémon/item you wish to find to find that Pokémon or the item. If you want to find a Pokémon with a specific gender, select the gender you wish to find.

Custom search by image code

Select By img code to find a Pokémon by img code. For example you wish to find a Bulbasaur. You paste it's Img code in the search bar:
//pfq-static.com/img/pkmn/1/g/g.png/t=1474027727
and now it will show you when a Bulbasaur is found! Copy paste the complete link (starting from //) or you won't find anything.
