Custom Search Help Custom search by Pokémon name

Select Custom Egg and/or Custom Pokémon and type the name of the Pokémon you wish to find to find that Pokémon or the egg of that Pokémon. It's case sensitive so use a capital letter. If you want to find a Pokémon with a specific gender, select the gender you wish to find.

Custom search by image code

Select By img code (and de-select Custom Egg & Custom Pokemon checkboxes) to find a Pokémon or egg by img code. For example you wish to find a Bulbasaur. You paste it's Img code in the search bar:
//pfq-static.com/img/pkmn/1/g/g.png/t=1474027727
and now it will show you when a Bulbasaur is found! Copy paste the complete link (starting from //) or you won't find anything.
